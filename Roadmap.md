# Roadmap

This is a short roadmap for the Responsive Fine-Tuner project.

Near-term (MVP)
- Stabilize core functionality (data ingestion, interactive labeling, LoRA-based fine-tuning)
- Improve documentation and tests
- Add example datasets and simple demos

Mid-term
- Add model registry and experiment tracking
- Add configurable deployment templates (Docker + Kubernetes)
- Improve scalability & distributed training

Long-term
- Auto-suggest labeling and active learning strategies
- Managed cloud deployment / enterprise integrations
