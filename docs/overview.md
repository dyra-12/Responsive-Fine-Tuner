## Overview

Responsive Fine-Tuner is an MVP for interactive, human-in-the-loop model fine-tuning.

This `docs/overview.md` contains a short description of the system, architecture, and how to run the demo.

Architecture
- `frontend/` — Gradio UI
- `backend/` — Model manager, data processor, training utilities

Run the demo locally:

```bash
python run_app.py
```
